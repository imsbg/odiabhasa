Common Questions (FAQs)

Everything you need to know about accessing textbooks via the Odia Bhasa App.

What is the Odia Bhasa App? +
The Odia Bhasa App is a dedicated educational platform providing Odisha Board students with free PDF access to Class 1 to 10 school textbooks without any ads or pop-ups.
Which Class 1 and Class 2 books are available on the Odia Bhasa App? +
Using the Odia Bhasa App, you can download Class 1 books like Jhulana - 1, Ganita Khela, and Hasakhela. For Class 2, we provide Jhulana - 2, Maja Majare Ganita, and Hasakhusi.
Can I download Class 3 and Class 4 books using the Odia Bhasa App? +
Yes! For Class 3, the Odia Bhasa App offers books like Bhasa Mahaka 1, Bichitra Ama Pruthibi, Ganita Mela, and Indradhanu. For Class 4, you can read Bhasa Mahaka 2, Ama Bichitra Biswa, and Sahitya Kalika completely free.
What books are included for Class 5 and Class 6 in the Odia Bhasa App? +
Class 5 students can find Bhasamahaka 3, Nabarasa, and Paribesa Bigyana on the Odia Bhasa App. For Class 6, books like Sahityasudha, Ganita Prakasha, Samajika Bigyana, and Kalakunja are available to download and read offline.
Does the Odia Bhasa App have textbooks for Class 7 and Class 8? +
Absolutely. The Odia Bhasa App library includes Class 7 titles like Sahityasumana, Jigyasa, and Kalakruti. For Class 8, you get free access to Sahityasurabhi, Kruti, Kousala Bodha, and Sarala Ganita.
Are Board Exam books for Class 9 and Class 10 available on the Odia Bhasa App? +
Yes, Board students can fully rely on the Odia Bhasa App! Download essential Class 9 and Class 10 books like Sahitya Sindhu, Madhyamika Byakarana, Bhoutika Bigyana, Jiba Bigyana, and Bhugola O Arthaniti for your exam preparations.
